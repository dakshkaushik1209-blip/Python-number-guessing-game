# Number Guessing Game 🎯

A simple Python console game where the player tries to guess a randomly generated number between 1 and 100.

## Features

* 3 Difficulty Levels

  * Easy (10 Attempts)
  * Medium (7 Attempts)
  * Hard (5 Attempts)
* Random number generation
* Input validation
* Range validation (1-100)
* Hints whether the guess is too high or too low
* Attempt counter
* Win/Lose conditions

## How to Play

1. Run the Python file.
2. Choose a difficulty level.
3. Guess a number between 1 and 100.
4. The program will tell you whether your guess is too high or too low.
5. Keep guessing until you find the correct number or run out of attempts.

## Technologies Used

* Python
* Random Module
